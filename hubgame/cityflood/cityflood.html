Game Logic Overview



Technical Features


Procedural Spawning Algorithm
Implemented a dynamic spawner using Random Float in Range and Vector Math. The system automatically generates obstacles and platforms ahead of the player, ensuring a non-repetitive environment by calculating randomized offsets for each spawned actor.

Character Movement & Sprite Flipping
Developed the player controller logic to handle horizontal movement and jumping. To ensure visual consistency, I implemented a Scale-based Flipping system that mirrors the character sprite in real-time based on movement input direction.

Object Cleanup & Resource Management
To optimize performance for an endless game, I developed a cleanup system that detects off-screen platforms using Collision Tags. This logic triggers the Destroy Actor sequence to free up memory and maintain a stable frame rate (FPS).
